# UniappStructureTemplate

一个用于规范项目目录的基础 UniApp 模板，保持结构清晰、便于扩展，适合作为新项目的起点。

## 目录结构

请查看：`目录结构说明.md`

## 适用范围

- Android / iOS App
- 微信小程序
- H5
- 鸿蒙（后期可扩展）

## 技术栈

- UniApp
- JavaScript（非 TS）
- Vue2 / Vue3 兼容（条件编译）

## 使用方式

1. 使用 HBuilderX 打开项目目录
2. 根据目标平台选择运行或发行
3. 按需扩展 `pages/` 与 `subpackages/`

## 约定说明

- 通用目录职责详见 `目录结构说明.md`
- 请求层统一放在 `request/`
- 状态管理放在 `store/`
