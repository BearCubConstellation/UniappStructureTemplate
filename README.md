# UniappStructureTemplate

一个面向公司级项目的 UniApp 基础模板，提供清晰的目录结构与可扩展的工程约定，适用于 Android / iOS / 微信小程序等多端项目。

## 目录结构

请查看：`目录结构说明.md`

## 适用范围

- Android / iOS App
- 微信小程序
- H5
- 鸿蒙（后期可扩展）

## 技术栈

- UniApp
- JavaScript（非 TS）
- Vue2 / Vue3 兼容（条件编译）

## 使用方式

1. 使用 HBuilderX 打开项目目录
2. 根据目标平台选择运行或发行
3. 按需扩展 `pages/` 与 `subpackages/`

## 约定说明

- 通用目录职责详见 `目录结构说明.md`
- 请求层统一放在 `request/`
- 状态管理放在 `store/`
