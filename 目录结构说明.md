# 目录结构说明

本文档用于说明 `UniappStructureTemplate` 的目录结构与职责，便于团队统一约定与后续扩展。

## 目录树（当前实际结构）

```
UniappStructureTemplate/
├─ api/
├─ common/
├─ components/
├─ config/
├─ pages/
├─ platforms/
├─ request/
├─ static/
├─ subpackages/
├─ store/
├─ styles/
├─ utils/
├─ App.vue
├─ index.html
├─ main.js
├─ manifest.json
├─ pages.json
├─ uni.promisify.adaptor.js
└─ uni.scss
```

## 目录与文件说明（表格）

| 路径 | 类型 | 说明 |
| --- | --- | --- |
| `api/` | 目录 | 接口封装目录，按业务模块拆分（如 `auth.js`、`user.js`）。 |
| `common/` | 目录 | 公共资源与常量目录（枚举、常量、全局配置、通用样式片段）。 |
| `components/` | 目录 | 通用组件目录，建议每个组件单独子目录并附使用说明。 |
| `config/` | 目录 | 环境与项目配置目录（dev/test/prod、开关、常量等）。 |
| `pages/` | 目录 | 页面路由目录，按 `pages/模块/页面.vue` 组织。 |
| `platforms/` | 目录 | 平台差异化实现目录（H5/小程序/App）。 |
| `request/` | 目录 | 统一请求层目录（封装请求、拦截器、鉴权、错误码处理）。 |
| `static/` | 目录 | 静态资源目录，打包时原样拷贝。 |
| `subpackages/` | 目录 | 子包页面目录（与 `pages.json` 的 `subPackages` 配置对应）。 |
| `store/` | 目录 | 状态管理目录（如 Pinia/Vuex 模块）。 |
| `styles/` | 目录 | 全局样式目录（reset、mixins、主题样式等）。 |
| `utils/` | 目录 | 工具函数目录（时间、格式化、校验、缓存等）。 |
| `App.vue` | 文件 | 应用级入口组件，应用生命周期与全局逻辑挂载处。 |
| `index.html` | 文件 | H5 入口模板，通常无需频繁修改。 |
| `main.js` | 文件 | 应用启动入口，兼容 Vue2/Vue3。 |
| `manifest.json` | 文件 | UniApp 核心配置（平台、权限、打包等）。 |
| `pages.json` | 文件 | 页面路由与全局样式配置。 |
| `uni.promisify.adaptor.js` | 文件 | Promise 化拦截器，统一处理 `uni.*` API 返回。 |
| `uni.scss` | 文件 | UniApp 内置样式变量（颜色、字号、间距等）。 |

## 维护建议（可选）

| 建议项 | 说明 |
| --- | --- |
| 编码统一 | 建议统一为 UTF-8，避免注释乱码。 |
| 文档维护 | 新增页面/目录时同步更新本说明文档。 |
| 模块说明 | 为 `components/` 与 `api/` 子模块补充 `README.md`。 |

## 多端项目结构补充建议（JS，无多语言）

| 分类 | 目录 | 说明 |
| --- | --- | --- |
| 更必要 | `hooks/` | 通用组合式逻辑（分页、列表、权限校验、加载态）。 |
| 更必要 | `assets/` | 构建参与处理的资源（svg、字体），与 `static/` 区分。 |
| 更必要 | `plugins/` | 第三方插件初始化与全局注册（埋点、推送、地图、分享）。 |
| 更必要 | `request/` | 请求层已存在，建议在团队规范中明确其职责与边界。 |
| 多端常用 | `platforms/app-plus/` | App 端差异化实现与配置片段。 |
| 多端常用 | `platforms/mp-weixin/` | 微信小程序端差异化实现与配置片段。 |
| 多端常用 | `platforms/h5/` | H5 端差异化实现与配置片段。 |
| 多端常用 | `platforms/harmony/` | 鸿蒙端差异化实现与配置片段（后期可加）。 |
| 多端常用 | `config/dev.js` | 开发环境配置。 |
| 多端常用 | `config/test.js` | 测试环境配置。 |
| 多端常用 | `config/prod.js` | 生产环境配置。 |
| 多端常用 | `mock/` | 联调期模拟数据（可选）。 |
